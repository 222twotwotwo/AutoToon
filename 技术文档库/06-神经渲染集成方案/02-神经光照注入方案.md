# 方案二：神经光照注入

## 方案概述

神经光照注入在方案一的基础上，进一步在ToonBxDF中注入神经光照，提供更强的风格控制能力。它保留Ramp采样，同时注入额外的神经光照效果。

## 详细设计

### 1. 神经光照注入器 (Neural Light Injector)

#### 网络架构

```python
import torch
import torch.nn as nn

class NeuralLightInjector(nn.Module):
    def __init__(self, style_dim=512, hidden_dim=1024):
        super().__init__()
        
        # 条件编码：将几何信息编码为特征
        self.condition_encoder = nn.Sequential(
            nn.Linear(12, hidden_dim),  # N, V, L, NoL, roughness, metallic
            nn.LayerNorm(hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim),
            nn.LayerNorm(hidden_dim),
            nn.ReLU()
        )
        
        # 融合层：融合风格特征和条件特征
        self.fusion = nn.Sequential(
            nn.Linear(style_dim + hidden_dim, hidden_dim * 2),
            nn.LayerNorm(hidden_dim * 2),
            nn.ReLU(),
            nn.Linear(hidden_dim * 2, hidden_dim),
            nn.LayerNorm(hidden_dim),
            nn.ReLU()
        )
        
        # 光照预测头
        self.light_head = nn.Sequential(
            nn.Linear(hidden_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, 512),
            nn.ReLU(),
            nn.Linear(512, 256),
            nn.ReLU(),
            nn.Linear(256, 3)  # RGB光照
        )
        
        # 混合权重头
        self.blend_head = nn.Sequential(
            nn.Linear(hidden_dim, 64),
            nn.ReLU(),
            nn.Linear(64, 1),
            nn.Sigmoid()  # [0, 1]
        )
    
    def forward(self, style_feature, geometry_feature):
        """
        style_feature: [B, style_dim]
        geometry_feature: [B, 12] - N, V, L, NoL, roughness, metallic
        """
        # 编码几何条件
        cond_feat = self.condition_encoder(geometry_feature)  # [B, hidden_dim]
        
        # 融合风格和条件
        fused = torch.cat([style_feature, cond_feat], dim=-1)  # [B, style_dim + hidden_dim]
        fused = self.fusion(fused)  # [B, hidden_dim]
        
        # 预测神经光照
        neural_light = self.light_head(fused)  # [B, 3]
        
        # 预测混合权重
        blend_weight = self.blend_head(fused)  # [B, 1]
        
        return {
            'neural_light': neural_light,
            'blend_weight': blend_weight
        }
```

#### 几何特征编码

```python
def encode_geometry_feature(N, V, L, NoL, roughness, metallic):
    """
    编码几何特征为12维向量
    """
    # N: [B, 3] - 法线
    # V: [B, 3] - 视线方向
    # L: [B, 3] - 光照方向
    # NoL: [B, 1] - N·L
    # roughness: [B, 1] - 粗糙度
    # metallic: [B, 1] - 金属度
    
    # 计算半角向量
    H = normalize(V + L)  # [B, 3]
    
    # 计算NoH
    NoH = torch.sum(N * H, dim=-1, keepdim=True)  # [B, 1]
    
    # 计算VoH
    VoH = torch.sum(V * H, dim=-1, keepdim=True)  # [B, 1]
    
    # 拼接所有特征
    geometry_feature = torch.cat([
        N,                    # 3
        V,                    # 3
        L,                    # 3
        NoL,                  # 1
        roughness,            # 1
        metallic,             # 1
        NoH,                  # 1
        VoH                   # 1
    ], dim=-1)  # [B, 14]
    
    return geometry_feature
```

### 2. 完整的端到端模型

```python
class HybriToonLightInjector(nn.Module):
    def __init__(self, style_dim=512, num_ramps=32):
        super().__init__()
        
        self.style_encoder = StyleEncoder(style_dim)
        self.ramp_predictor = NeuralRampPredictor(style_dim, num_ramps)
        self.light_injector = NeuralLightInjector(style_dim)
    
    def forward(self, style_image, geometry_feature):
        # 编码风格
        style_feature = self.style_encoder(style_image)
        
        # 预测Ramp
        ramp_predictions = self.ramp_predictor(style_feature)
        
        # 注入神经光照
        light_predictions = self.light_injector(style_feature, geometry_feature)
        
        return {
            'ramp': ramp_predictions,
            'light': light_predictions
        }
```

    // 2. 高光计算（原有）

    // 3. 描边计算（原有）

#if HYBRITOON_ENABLED
    // 编码几何特征

    // 获取神经光照
    float3 NeuralLight = HybriToon.NeuralLight;
    float BlendWeight = HybriToon.BlendWeight;
    
    // 混合原有光照和神经光照
    Diffuse = lerp(Diffuse, Diffuse * NeuralLight, BlendWeight);
    Specular = lerp(Specular, Specular * NeuralLight, BlendWeight);
#endif
    // ===========================================
    
    return FDirectLighting(Diffuse + Specular + Outline, 1.0);
}

### 2. 损失函数

```python
def compute_loss(
    ramp_predictions,
    light_predictions,
    rendered_image,
    target_image,
    target_diffuse_ramp_idx,
    target_diffuse_uv_offset,
    target_specular_ramp_idx,
    target_specular_uv_offset
):
    # Ramp损失（同方案一）
    diffuse_ramp_loss = F.cross_entropy(
        ramp_predictions['diffuse_ramp_idx'],
        target_diffuse_ramp_idx
    )
    specular_ramp_loss = F.cross_entropy(
        ramp_predictions['specular_ramp_idx'],
        target_specular_ramp_idx
    )
    diffuse_uv_loss = F.l1_loss(
        ramp_predictions['diffuse_uv_offset'],
        target_diffuse_uv_offset
    )
    specular_uv_loss = F.l1_loss(
        ramp_predictions['specular_uv_offset'],
        target_specular_uv_offset
    )
    
    # 神经光照损失
    neural_light = light_predictions['neural_light']
    blend_weight = light_predictions['blend_weight']
    
    # 图像重建损失
    perceptual_loss = compute_perceptual_loss(rendered_image, target_image)
    
    # 光照平滑性损失
    light_smoothness_loss = torch.mean(torch.abs(neural_light[:, 1:] - neural_light[:, :-1]))
    
    # 总损失
    total_loss = (
        diffuse_ramp_loss +
        specular_ramp_loss +
        0.1 * diffuse_uv_loss +
        0.1 * specular_uv_loss +
        1.0 * perceptual_loss +
        0.01 * light_smoothness_loss
    )
    
    return total_loss
```

## 优缺点总结

### 优点 ✅

1. **效果比方案一更好**
   - 可以注入复杂的光照效果
   - 不受限于Ramp纹理
   - 更灵活的风格控制

2. **保持大部分MooaToon管线**
   - 描边、头发阴影等都保留
   - 可以混合使用神经和传统光照

3. **可以渐进式改进**
   - 先用方案一
   - 再升级到方案二
   - 不需要重写整个系统

### 缺点 ❌

1. **修改量中等**
   - 需要修改ToonBxDF
   - 需要集成光照注入器
   - 调试更复杂

2. **性能影响中等**
   - 光照注入器需要每像素推理
   - 可能影响帧率
   - 需要优化

3. **需要更多调试**
   - 光照注入可能引入伪影
   - 需要调整混合权重
   - 可能需要更多超参数

## 文档导航指南

### 如果你是零基础开发者

**推荐阅读顺序**：
1. 先看技术文档库外的 `MooaToon开发前置知识-图形学与TA基础.md`
2. 然后看 `07-技术报告汇总/` 里的4份技术报告
3. 最后根据兴趣深入 `02-05` 的源码分析

---

### 如果你是有基础的开发者

**推荐阅读顺序**：
1. 先看 `06-系统架构与设计/01-系统架构总览.md`
2. 再看 `06-系统架构与设计/02-数据流分析.md`
3. 根据你的工作方向选择深入：
   - **引擎开发**：看 `02-引擎核心分析/` 和 `05-渲染管线分析/`
   - **材质/TA**：看 `03-材质系统分析/` 和 `04-着色器源码分析/`
   - **HybriToon/AI**：看 `06-系统架构与设计/04-HybriToon架构设计.md`

---

### 如果你想快速上手

**推荐**：
1. 看 `07-技术报告汇总/技术详细报告4-系统级深度分析.md`
2. 看 `01-前置知识与入门/04-TA基础.md`
3. 开始做第一个Toon材质

## 关键概念速查表

### GBuffer相关

| 概念 | 说明 |
|------|------|
| **GBuffer** | Geometry Buffer，存储几何信息的缓冲区 |
| **ToonBufferA** | MooaToon新增的GBuffer，存储Toon专用数据 |
| **FToonGBufferData** | Toon渲染数据结构体 |

### 材质属性相关

| 概念 | 说明 |
|------|------|
| **MP_MooaEncodedAttribute0-4** | 5个Float4材质属性，传递Toon参数 |
| **MSM_Toon** | Toon着色模型枚举 |
| **MATERIAL_SHADINGMODEL_TOON** | 着色器编译宏 |

### 渲染相关

| 概念 | 说明 |
|------|------|
| **BasePass** | 基础通道，渲染GBuffer |
| **DeferredLighting** | 延迟光照，计算光照 |
| **ToonBxDF** | Toon双向反射分布函数，核心着色逻辑 |
| **Ramp纹理** | 渐变纹理，卡通渲染核心 |

## 代码注释约定

MooaToon 源码中的注释非常友好，主要使用中文：

| 注释格式 | 含义 |
|---------|------|
| `// Mooa Xxx` | MooaToon 新增/修改的代码 |
| `// Mooa Toon Shading Model` | 着色模型相关 |
| `// Mooa Anisotropy` | 各向异性相关 |
| `// Mooa Ray Tracing Shadow` | 光线追踪阴影相关 |
| `// Mooa End` | MooaToon 修改结束 |
